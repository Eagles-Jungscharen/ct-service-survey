# Deployment Scripts

Dieses Verzeichnis wird PowerShell-Scripts für Deployment-Prozesse enthalten.

## Geplante Scripts

- `deploy.ps1` - Deploy Infrastructure (Bicep), schreibt infrastructure.local
- `deploy-code.ps1` - Deploy Frontend/Backend Code
- `setup-custom-domains.ps1` - Optional: Custom Domain Setup
